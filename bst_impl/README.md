## Задача

Реализовать основные операции бинарного дерева поиска (BST):

- **Класс `TreeNode`** — реализует структуру узла дерева.
- **Класс `BinarySearchTree`** с методами:
    - `insert(key, value)` – вставка элемента;
    - `search(key)` – поиск элемента;
    - `delete(key)` – удаление элемента;
    - `height()` – вычисление высоты дерева;
    - `is_balanced()` – проверка сбалансированности дерева.

---

### Технические требования

- Python **3.8+**
- Используйте **UV** для управления зависимостями
- Добавьте инструкцию по запуску и проверке методов

---

## Инструкция по запуску и проверке

### Запуск тестов

Для проверки работы всех методов бинарного дерева поиска запустите тесты:

```bash
python test_tree.py
```

Тесты проверяют:
- Базовые операции (вставка, поиск, удаление, высота, сбалансированность)
- Сложные сценарии (несбалансированные деревья, удаление корня, граничные случаи)

### Пример использования

```python
from tree import BinarySearchTree

# Создание дерева
bst = BinarySearchTree()

# Вставка элементов
bst.insert(50, "root")
bst.insert(30, "left")
bst.insert(70, "right")

# Поиск элемента
value = bst.search(30)  # Вернет "left"

# Высота дерева
height = bst.height()  # Вернет 1

# Проверка сбалансированности
is_balanced = bst.is_balanced()  # Вернет True

# Удаление элемента
bst.delete(30)
```
