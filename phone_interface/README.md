## Задача

Реализуйте абстракцию с использованием классов. Класс должен задавать шаблон (абстрактный тип данных) для объекта "**Телефон**".

### Требования:

1. **Создайте класс `Phone`.**
2. В методе `__init__` определите атрибуты, которые характеризуют телефон:
    - `brand` (`str`) — бренд
    - `model` (`str`) — модель
    - `price` (`float`) — цена
    - `color` (`str`) — цвет
    - `storage_gb` (`int`) — объём памяти в гигабайтах
    - `is_in_stock` (`bool`) — наличие в магазине

   Используйте *аннотации типов* (type hints) для всех атрибутов.

3. Реализуйте методы, которые предоставляют интерфейс для работы с телефоном:
    - `get_full_name() -> str` — возвращает строку в формате `"Бренд Модель"`.
    - `apply_discount(self, discount_percent: float) -> None` — уменьшает цену телефона на заданный процент.
    - `check_availability(self) -> str` — возвращает сообщение `"В наличии"` или `"Нет в наличии"` в зависимости от поля `is_in_stock`.

4. Реализуйте метод `__str__` для красивого строкового представления объекта, чтобы при печати через `print()` можно было получить развёрнутую информацию о телефоне.

5. Создайте **3-4 экземпляра** класса `Phone` с разными данными и продемонстрируйте работу всех методов на этих объектах.
